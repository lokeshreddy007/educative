Removing Object Properties

In this lesson, we will see how to remove certain properties from an object.

We'll cover the following
    • The delete construct

The delete construct #

JavaScript provides the delete construct to remove properties from objects. This code demonstrates the usage:

var myObject = new Object();
myObject.intProp = 23;
myObject.strProp = "Hi!";
console.log(myObject.intProp); // 23
console.log(myObject.strProp); // Hi!
delete myObject.intProp;
console.log(myObject.intProp); // undefined
console.log(myObject.strProp); // Hi!
Enter to Rename, Shift+Enter to Preview

The code sets up the myObject instance with two properties, intProp and strProp. The delete statement removes intProp from myObject so that the second time it is written to the console it contains undefined.
