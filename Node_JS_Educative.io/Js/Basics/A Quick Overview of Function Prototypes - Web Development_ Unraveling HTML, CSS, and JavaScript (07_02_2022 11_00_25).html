A Quick Overview of Function Prototypes

In this lesson, we'll cover the function prototyping in JavaScript. Let's begin!

We'll cover the following
    • Listing 7-14: Creating functions with the prototype property
      • Explanation

Functions in JavaScript, including constructor functions, are created with a prototype property, which is an object containing properties and methods that should be available to instances of a particular object type. This object is utilized when creating new object instances, and all of its properties and methods are shared among those object instances.

Take a look at Listing 7-14 below:

Explanation#

This code adds two properties (wheels and name) and a function (getWheels) to the prototype property of Car. When resolving property names, the JavaScript engine examines not only the object instance, but also the constructor functions prototype. As a result of this behavior, the car1 and car2 objects will have all properties added to the prototype.
